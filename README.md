# 🏸 Pro Badminton Scorer

A premium, modern badminton scoring application built entirely using **Google Antigravity** and **Gemini**. This project demonstrates the power of "Vibe Coding" — creating functional, aesthetically pleasing web applications through natural language conversation with AI. Of course I had to steer a couple of times when it went wrong or broke some features, which goes to show it isnt completely perfect, but gets the job done without me writing a single line of code. 

## 🚀 How it Works

This project was built from scratch without manual coding of the logic or styles. Every line of code, every CSS transition, and the entire game logic were generated by **Antigravity**, an agentic AI coding assistant.

The development process involved:
1. **Natural Language Specification**: Describing features and design preferences.
2. **Iterative Refinement**: Debugging and adding features (like Undo and Match History) through conversation.
3. **Automated Tool Usage**: Antigravity used tools like `generate_image` (for assets) and direct file manipulation to build the app.

## ✨ Features

- **🏆 Real-time Scoring**: Tracks points with animations and follows official badminton rules (Deuce, Golden Point at 29-29).
- **🎲 Server Randomizer**: Fairly decide who starts the game with a single click.
- **⎌ Smart Undo**: Accidentally added a point? Quickly revert the last action at any time, even from the winner screen.
- **📊 Match History**: Persistently tracks your session's match results and provides win statistics.
- **✨ Premium Aesthetics**: Uses a sleek Glassmorphism design system, modern typography (Outfit), and smooth micro-animations.
- **🎉 Victory Celebration**: Custom confetti effects powered by `canvas-confetti` when a match is won.
- **📱 Responsive Layout**: Optimized for both desktop and mobile use.

## 🛠️ Tools & Technologies Used

- **Google Antigravity**: The AI agent that orchestrated the build.
- **Gemini**: The underlying Large Language Model (LLM) powering the intelligence.
- **HTML5 & Vanilla CSS**: For structure and high-performance styling.
- **Modern JavaScript**: For state management and game logic.
- **Canvas Confetti**: For the victory celebration.
- **Google Fonts**: Specifically the 'Outfit' typeface for a premium feel.
